## 🐛 Troubleshooting

### Website doesn't load
- Check console for errors (F12 in browser)
- Verify `data.json` is valid JSON (use jsonlint.com)
- Ensure running from HTTP server (not `file://`)

### Data not updating
- Hard refresh browser (Cmd+Shift+R / Ctrl+Shift+R)
- Clear browser cache
- Check `data.json` syntax

### Images not showing
- Profile photo URL needs to be accessible
- Consider adding images to `assets/` folder
- Update image paths in HTML/data.json
